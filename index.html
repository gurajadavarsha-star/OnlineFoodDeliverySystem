Online Food Delivery System

A Java-based Online Food Delivery System developed using Data Structures and Algorithms.

Features

  • Add Restaurants
  • Display Restaurants
  • Sort Restaurants by Rating
  • Find Delivery Route using Dijkstra Algorithm
  • Assign Orders
  • Maximize Profit using Dynamic Programming

Technologies Used

  • Java
  • Data Structures
  • Dijkstra Algorithm
  • Dynamic Programming
  • Object-Oriented Programming
