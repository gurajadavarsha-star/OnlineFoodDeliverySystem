# Online Food Delivery System

A Java-based Online Food Delivery System implementing:

## Features
- Restaurant Management
- Restaurant Sorting by Rating
- Delivery Route Optimization using Dijkstra Algorithm
- Order Scheduling
- Profit Maximization using Dynamic Programming

## Technologies Used
- Java
- Collections Framework
- Dijkstra Algorithm
- Dynamic Programming
- Object-Oriented Programming
